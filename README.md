# 🍽️ CodeHelp Dhabba – Food Recipe UI

A modern and responsive food recipe website built using **HTML and CSS**. This project showcases a clean user interface with interactive hover effects and a structured layout for displaying food items and their recipes.

---

## 📌 Features

* 🍛 Beautiful food item cards with images
* 📖 Ingredients and recipe sections
* 🎨 Modern UI with gradients and shadows
* ✨ Smooth hover effects with overlay interaction
* 📱 Responsive and clean layout
* 🔝 Fixed navigation bar

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Gradients, Transitions)
